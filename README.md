# Horror Maze

OpenGL CGV Project. Procedural maze, Phong shading, BFS enemy AI.

## Build
CMake handles GLFW and GLM automatically.
1. Clone repo
2. Open in VS Code
3. Run `CMake: Configure`
4. Build/Run

## Git Rules
* Use your branch: `p1/maze`, `p2/assets`, `p3/audio`, `p4/enemy`, `p5/shaders`
* Merge twice daily minimum.
* Person 4 (Me) handles all merge conflicts to `main`. Don't force push.

## Roles
* **P1:** Maze generation, collision
* **P2:** Assimp models, textures
* **P3:** Audio (irrKlang)
* **P4:** Game state, Enemy AI (BFS)
* **P5:** Shaders, post-processing
